INSTALL:
########

Step 01: 	Download the latest monoto version from github
			Stable: 	https://github.com/downloads/macfidelity/monoto/monoto_20120707.04_milestone01.zip
			Unstable: 	https://github.com/macfidelity/monoto/zipball/master

Step 02: 	Extract the archive on disk

Step 03: 	Check conf/config.php and modify it to your needs (mysql settings)

Step 04: 	Prepare your mysql server
			- create databases (using the informations from doc/MYSQL_STRUCT.txt)

Step 05: 	upload the extracted files to your webserver target url

Step 06: 	visit setup.php in your browser and create your admin user with this script.

Step 07: 	Secure your monoto install
			- conf/config.php (only webserver should be allowoed to read this file)
			- remove/delete setup.php
			- optional: add .htaccess protection to your monoto installation - as it isnt designed for security aspects right now. Sorry

Enjoy!
